Editor notes

Use block titles for natural transitions like Therefore, Hence, or When n is even. Keep the structural type small and stable.

Save order follows the numeric Order field. The live preview uses that sequence even if the editor cards themselves are still stacked in their original order.

The preview renders LaTeX-style math with the same MathJax stack used elsewhere in the project. In Body format: LaTeX, you can paste an image from the clipboard into Body source to upload it and insert \includegraphics automatically.

Download PDF typesets your last saved blocks with Evan Chen’s evan.sty (KOMA class) on the server; fix compile errors shown in the log if export fails.
