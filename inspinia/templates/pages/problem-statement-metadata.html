Export metadata workbook

Download all statement rows as an .xlsx workbook with editable TOPIC, MOHS, Confidence, IMO slot guess, and Topic tags columns, plus identity fields (CONTEST YEAR, CONTEST NAME, DAY LABEL, PROBLEM NUMBER, PROBLEM CODE, and STATEMENT LATEX) for context while you edit.

On import, each row updates the statement identified by STATEMENT UUID (with PROBLEM UUID as a legacy fallback). Blank import rows are ignored, and blank cells keep existing values. Identity columns are exported for context only and are ignored on import, so you can fill many lines in one workbook and upload once.
