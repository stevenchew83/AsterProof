Export metadata workbook

Download all statement rows as an .xlsx workbook with editable link columns (LINKED PROBLEM UUID) and metadata columns: TOPIC, MOHS, Confidence, IMO slot guess, Topic tags, Core ideas, Rationale, and Common pitfalls. It also includes LINK STATUS, LINKED PROBLEM LABEL, and identity fields for context while you edit.

On import, each row updates the statement identified by STATEMENT UUID (with PROBLEM UUID as a legacy fallback). Filled identity cells can update the statement row, and a filled LINKED PROBLEM UUID links or relinks the tracked problem row. Blank import rows are ignored, and blank cells keep existing values.

Import metadata

This updates each ContestProblemStatement row by STATEMENT UUID, links or relinks by LINKED PROBLEM UUID, upserts the linked ProblemSolveRecord metadata, and keeps the link in sync automatically.

Use this when you want to bulk update metadata for multiple statement rows without editing them one by one. Upload a workbook or paste TSV/CSV rows with a header row. Blank cells keep existing values, while a new linked problem UUID creates a tracked problem row when TOPIC and MOHS are present. Pitfalls is also accepted as an import header for common pitfalls.
